SForecast: Enterprise forecasting platform for packaging/manufacturing. Features multi-material forecasting, unit conversion (kg/liters/sqm), financial analysis with currency support, demand planning, Built with Streamlit, Prophet ML models, SQLite database. Enables accurate inventory optimization, cost forecasting, and production planning.
